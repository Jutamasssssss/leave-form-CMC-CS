กรุณาอ่านและทำความเข้าใจเงื่อนไขการลาให้ครบถ้วนก่อนยื่นคำขอ
เอกสารนี้จัดทำขึ้นเพื่อใช้ภายในเท่านั้น

เลื่อนอ่านเงื่อนไขทั้งหมดก่อนกดยอมรับ
01
การจัดตารางการทำงานรายเดือน
จะพิจารณาจากปริมาณงานและจำนวนบุคลากรเป็นหลัก เพื่อให้สอดคล้องและเกิดประสิทธิภาพสูงสุด
02
ลากิจ / ลาพักร้อน / ลาป่วยล่วงหน้า (มีเอกสารประกอบ)
  1. ต้องแจ้งล่วงหน้าก่อนวันลาจริง — ลากิจต้องมีเอกสารประกอบเสมอ
  2. ต้องได้รับการอนุมัติก่อน จึงจะดำเนินการลาผ่านระบบ Prosoft ได้
  3. ลากิจ / ลาพักร้อน ต้องแจ้งล่วงหน้าอย่างน้อย 7 วันทำการ — ลาพักร้อนที่อยู่ในเงื่อนไข ไม่ต้องแนบเอกสาร
03
ลาป่วย (มีผลบังคับใช้ทันที)
  1. โทรแจ้งโดยตรงล่วงหน้า 2 ชั่วโมงก่อนเวลาเข้างาน
  2. หากไม่รับสาย ให้ส่ง LINE ส่วนตัว พร้อมเก็บ Screenshot
  3. หากไม่มีหลักฐาน ถือว่าเป็น ขาดงาน
  4. แนบหลักฐานผ่านระบบหลังกลับมาทำงานวันถัดไป
04
ลาหยุดชดเชยวันหยุดราชการ
  1. ต้องใช้สิทธิ์ภายใน 2 เดือน นับจากวันหยุดดังกล่าว
  2. หากไม่สามารถใช้สิทธิ์ภายใน 2 เดือนได้ ต้องขออนุมัติจาก Manager หรือ Assistant Manager
ตัวอย่าง: 13 ต.ค. 2568 เป็นวันหยุดราชการแต่ทำงาน → ขอชดเชยได้ตั้งแต่ 14 ต.ค. ถึง 14 ธ.ค. 2568
05
การลาอื่นๆ นอกเหนือเงื่อนไขข้างต้น
กรุณาติดต่อ Manager และ Assistant Manager โดยตรงเท่านั้น
